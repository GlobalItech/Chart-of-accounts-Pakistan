Chart of Accounts Pakistan

Chart of Accounts to facilitate Financial Reporting as per IFRS and other applicable local standards, having Raw Material, WIP & Finished Goods Accounts for manufacturing concerns. There are additional account types other than default Odoo account types for financial reports. Parent accounts can be created for multi-level Chart of Accounts. Works for both Odoo10 Community and Enterprise versions.

Installing Instructions.
If you created database with country selection install this COA before installation the Odoo default Accounting module. If no country selected at the time of creating new database then you can install this module at any time but make sure no other COA is installed before that.
